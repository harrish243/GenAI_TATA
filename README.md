# GenAI Powered Data Analytics Job Simulation

This repository contains materials and tasks for a GenAI-powered data analytics job simulation project.

## Project Structure

### Material/
This folder contains educational materials and resources:
- **Common techniques for credit risk modeling.txt** - Overview of credit risk modeling approaches
- **Evaluating model performance.txt** - Methods for assessing machine learning model performance
- **How GenAI assists in predictive mod.txt** - Role of Generative AI in predictive modeling
- **Key steps in conducting EDA.txt** - Essential steps for Exploratory Data Analysis
- **Utilizing GenAI tools to generate m.txt** - Guide to using GenAI tools for model generation
- **Tata_Data_Analytics_Glossary (1).docx** - Comprehensive glossary of data analytics terms

### TASK/
Contains practical tasks and datasets:
- **Task_1.ipynb** - First task notebook
- **Task_1 Dataset/** - Dataset and example answers for Task 1
- **Task_2/** - Second task materials including:
  - Task_2.ipynb - Second task notebook
  - Model planning templates and example answers

## Getting Started

1. Clone this repository
2. Install required dependencies (if any)
3. Start with the Material folder to understand the concepts
4. Complete the tasks in the TASK folder

## Prerequisites

- Python 3.x
- Jupyter Notebook
- Required Python packages (install via pip if needed)

## Usage

1. Open the Jupyter notebooks in the TASK folder
2. Follow the instructions in each notebook
3. Use the materials in the Material folder as reference
